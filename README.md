# Modern React Portfolio

A modern, professional, responsive, and visually polished personal portfolio website built with React.js and Vite.

## Features

- **React.js Only**: Built with functional components and hooks.
- **Modern UI**: Clean spacing, soft shadows, glassmorphism, and a professional dark theme default.
- **Easy to Update**: All content (projects, skills, certificates) is driven by a single `src/data/portfolioData.js` file.
- **Responsive**: Fully optimized for mobile, tablet, and desktop devices.
- **Animations**: Smooth transitions and fade-ins powered by Framer Motion.

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment to Vercel

This project is optimized for deployment on Vercel.

1. Push your code to a GitHub repository.
2. Go to [Vercel](https://vercel.com/) and sign in.
3. Click on "Add New" > "Project".
4. Import your GitHub repository.
5. Vercel will automatically detect that this is a Vite project and configure the build settings (`npm run build`, `dist` output directory).
6. Click "Deploy".
